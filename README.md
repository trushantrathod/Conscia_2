# Conscia: An AI-Powered Ethical Shopping Companion

Conscia is a full-stack application designed to enhance
productivity through intelligent automation and seamless browser
integration. It combines a custom AI-powered backend with a Chrome
Extension interface for real-time interactions.
Our Live Website - https://consciashop.netlify.app/
------------------------------------------------------------------------

## ✨ Features

-   AI-powered processing for intelligent outputs\
-   Chrome Extension for quick browser access\
-   Conversational and context-aware interaction\
-   Real-time API-based actions\
-   Clean and minimal UI

------------------------------------------------------------------------

## 🧠 System Architecture

-   Data Processing Layer\
-   Neural Network Model Layer\
-   Flask Backend APIs\
-   Frontend + Chrome Extension

------------------------------------------------------------------------

## 🔁 How It Works

1.  User interacts via UI or extension\
2.  Request sent to backend\
3.  AI processes input\
4.  Response returned instantly

------------------------------------------------------------------------

## 🧩 Chrome Extension

-   Toolbar-based quick actions\
-   No need to open main app\
-   Fast backend communication

------------------------------------------------------------------------

## 🛠️ Tech Stack

Backend: - Python, Flask\
- TensorFlow / Keras\
- Scikit-learn, Pandas

Frontend: - React.js

Extension: - Chrome Extension APIs

2.  Enable Developer Mode\
3.  Click Load Unpacked\
4.  Select extension folder
